# Lexxy

Lisp, but in Hex.

Reference: https://norvig.com/lispy.html

## Types

| Scheme     | Lex              |
| ---------- | ---------------- |
| symbol     | pattern          |
| number     | number           |
| atom       | symbol \| number |
| list       | list             |
| expression | atom \| list     |
| env        | list             |
